# Questionnaire Responses: wheat__pathogen_panel

- Questionnaire ID: agronomy-core-v1

## reusable_benchmark

- Status: answered

- A reusable benchmark is a stage-specific pathogen pressure panel tied to emergence, canopy closure, and flowering.
- A companion diagnostic is a standardized sampling + assay bundle for dominant fungal pressure windows.

### Key Claims

- Stage-specific pathogen pressure panels can be standardized across wheat trials. (support=direct, evidence=E100, E101)

## recurring_monitoring

- Status: partial

- Recurring monitoring is most defensible during vegetative and reproductive windows.
- Frequency should be tied to disease pressure and weather transitions rather than a fixed universal cadence.

### Key Claims

- Monitoring cadence should follow disease pressure windows rather than a single fixed schedule. (support=partial, evidence=E102)
